# Threads Social Strategist for Zo

A portable Zo skill for creating daily Threads content, auditing weekly performance, and planning useful replies. It works for creators, founders, operators, and brands without assuming a particular industry or audience.

## Install on Zo

Open Zo's Terminal and run:

```bash
cd /home/workspace
git clone https://github.com/stxxvibecode/threads-social-strategist.git Skills/threads-social-strategist
```

If the skill is already installed, update it with:

```bash
git -C /home/workspace/Skills/threads-social-strategist pull
```

## First Use

In Zo chat, tell the agent:

```text
Use the Threads Social Strategist skill for my account.

Handle: @yourhandle
Audience: [who you want to reach]
Objective: [leads, authority, event registrations, research participants, or another outcome]
Proof and topics: [your experience, products, services, events, or areas of expertise]
Voice: [how your posts should sound]
```

The skill will create a strategy based on that information rather than reusing another person's identity, business, or results.

## Daily Agent

Ask Zo to create a daily agent with this instruction:

```text
Use the installed Threads Social Strategist skill every morning at 9:00 AM in my timezone. Create seven distinct ready-to-post drafts, save them to a dated file, and do not publish anything automatically. Every Friday, audit the prior seven days before drafting the new posts. Use only available metrics and clearly mark any qualitative conclusions.
```

## What It Produces

- Seven daily Threads drafts across distinct content roles.
- Suggested posting times and a strategic purpose for each post.
- A weekly evidence-based audit and next-week test ideas.
- A reply map for conversations where the account can add real value.

It does not publish, reply, or send DMs without explicit approval.
